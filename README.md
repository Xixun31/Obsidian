# 📚 Obsidian Study Vault

這是我的個人學習筆記庫，主要記錄每日學習進度與學術研究內容。

## 📁 目錄結構

- `Study/`: 核心筆記空間（Obsidian Vault 根目錄）。
  - `YYYYMMDD.md`: 每日學習日誌，採時間序列記錄。
  - `.obsidian/`: 設定檔與外掛配置。

## 🛠️ 核心工具與外掛

本筆記庫配置了以下關鍵工具以提升效率：

1. **Obsidian Git**: 自動化版本控制，確保筆記不遺失並同步至雲端。
2. **LaTeX Suite**: 強化數學公式輸入體驗，適合學術與理科筆記。

## 💡 建議的進階工作流 (Anticipated Needs)

既然你已經用了 `obsidian-git` 和 `latex-suite`，我建議可以考慮以下擴充：

*   **MOC (Map of Content)**: 在 `Study/` 下建立一個 `Index.md`，用雙向連結彙整主題，避免日誌太過零散。
*   **Dataview 追蹤**: 建議安裝 Dataview 外掛，可以自動彙整每日筆記中的標籤（Tags）或進度。
*   **Kanban 視圖**: 針對長期學習計畫，可以使用 Kanban 外掛來視覺化你的 Study Roadmap。
